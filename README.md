# Pizzaria Maná

Landing page estática da Pizzaria Maná, do pizzaiolo Marcos Gomes de Souza.

## Conteúdo

- Página inicial em `index.html`
- Estilos em `styles.css`
- Animações leves em `script.js`
- Imagens e identidade visual em `Imagens/`

## Como abrir localmente

Abra o arquivo `index.html` no navegador ou sirva a pasta com qualquer servidor estático.
